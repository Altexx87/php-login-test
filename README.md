# PHP Login System

Simple login system for web applications with PHP & Bootstrap.

## Quick Start

1. Edit `dbcon.php` with your database details
2. Run `setup.sql` in your database
3. Upload files to your server
4. Visit `signup.php` to start

## Files

- `signup.php` - Create account
- `login.php` - Login
- `profile.php` - User profile
- `logout.php` - Logout
- `dbcon.php` - Database config (edit this!)
- `setup.sql` - Database table
